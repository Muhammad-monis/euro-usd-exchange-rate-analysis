# Euro-USD Storytelling Data Visualization

This project explores historical Euro to US Dollar exchange rates and presents insights using storytelling data visualization techniques.

## Project Goals
- Clean and prepare time-series exchange rate data
- Analyze trends using rolling averages
- Highlight key economic periods such as:
  - 2007–2008 financial crisis
  - Exchange rate trends under recent US presidencies
- Create professional-style visualizations inspired by FiveThirtyEight

## Tools Used
- Python
- Pandas
- Matplotlib

## Key Visualizations
- Long-term Euro-USD trend with rolling mean
- Financial crisis impact visualization
- Comparative trends across US presidential terms

## Files
- euro_exchange_analysis.ipynb – complete analysis and visualizations  
- euro-daily-hist_1999_2020.csv – exchange rate dataset

## Conclusion
This project demonstrates data cleaning, time series analysis, and storytelling visualization techniques for real-world financial data.
